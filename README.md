# Sparta_Adventure
[Unity 9기] 스파르타 던전 탐험 만들기

How To Play

이동

상: W, 하: S, 좌: A, 우: D
점프 Space_bar

단축키

인벤토리 : E
상호작용 및 습득 : F
공격 : 좌클릭
원거리 공격 : 우클릭 //원거리 무기 (돌)을 장착했을때만 사용가능
대쉬 : L쉬프트 (키다운)
